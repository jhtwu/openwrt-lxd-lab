OpenWrt LXD Test Lab

自動化、標準化、視覺化的在地化路由器測試系統

GitHub Repository

LXD 輕量化

利用 Linux 容器技術，秒級啟動 OpenWrt 與測試節點，資源佔用極低。

真實拓撲模擬

模擬 WAN/LAN 隔離環境，支援 NAT、防火牆與路由轉發實測。

CI/CD 整合

整合 GitHub Actions，每次 Commit 自動進行 ShellCheck 與功能連通性測試。

網路拓撲圖 (Topology)

[wan-host: 10.0.0.2] <--- (br-wan) ---> [openwrt: 10.0.0.1 | 192.168.1.1] <--- (br-lan) ---> [lan-host: 192.168.1.10]

自動化流水線 (CI/CD Pipeline)

1. 靜態語法檢查 (Lint)

使用 ShellCheck 確保所有自動化腳本符合標準規範，無語法隱患。

2. 虛擬環境佈建

在 GitHub Runner 中自動配置 LXD 服務，並建立隔離網橋。

3. 連通性實測

執行跨區段 Ping 測試與 Traceroute 追蹤，確保 Routing 與 NAT 運作正常。

快速部署

git clone git@github.com:jhtwu/openwrt-lxd-lab.git
sudo make all

此指令將自動完成網橋建立、容器部署與連通性測試。
